Acerca del proyecto

Este ejemplo muestra cómo publicar una web estática usando Docker Compose, Apache y un repositorio público de GitHub.

Flujo de despliegue

  1. Docker Compose levanta un contenedor temporal con Git.
  2. El contenedor clona el repositorio público.
  3. Copia el directorio site/ a un volumen Docker.
  4. Apache sirve ese volumen como raíz web.
